Stable documentation (0.1.x)

Install

npm install @projectyoked/expo-media-engine
npx expo prebuild

Requires a development build. Expo Go is not supported.

Requirements

  • Expo SDK 49+
  • expo-modules-core 1.0.0+
  • React Native 0.64+
  • iOS 13.4+, Android API 21+

API

  • isAvailable() — returns true if the native module is linked
  • extractAudio(videoUri, outputUri) — pulls audio from a video to .m4a
  • getWaveform(audioUri, samples?) — normalized RMS amplitude array (0–1)
  • exportComposition(config) — single-video export with text/emoji overlays and optional background music

exportComposition config

Required: videoPath, outputPath. Quality via quality ('low' | 'medium' | 'high') or an explicit bitrate in bps.

Overlay fields use parallel arrays: textArray, textX, textY, textColors, textSizes, textStarts, textDurations, and the equivalent emoji* fields. Audio mix: musicPath, musicVolume, originalVolume.

TypeScript

Types ship with the package. See ExportCompositionConfig in src/index.d.ts.

Looking for more features?

The pre-release line adds multi-track composition, real-time preview, filters, transitions, video stitching, compression, and full audio mixing. Install with npm install @projectyoked/expo-media-engine@alpha and read the pre-release documentation.
