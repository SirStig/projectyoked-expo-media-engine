Expo Media Engine

Open-source Expo native module for hardware-accelerated video composition and editing on iOS and Android.

Built on AVFoundation (iOS) and MediaCodec + OpenGL ES 2.0 (Android), the engine gives you real-time preview, multi-track composition, 9 filters, 8 transitions, audio mixing, text and image overlays, video stitching, and compression — without proprietary SDKs or per-minute billing.

The package is published on npm as @projectyoked/expo-media-engine. The current pre-release line (1.0.0-alpha, npm tag alpha) contains the full feature set. A legacy stable line (0.1.x, npm tag latest) covers basic single-video export and audio utilities.

Background

This library was built at Project Yoked LLC for the Project Yoked app (see the strip above). It exists because we needed native composition, preview, and related media behavior in Expo without vendor lock-in or prohibitive third-party video SDKs, so we implemented it in-house and released it under the MIT license.
